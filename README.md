# PEERLEARN
Students lack a centralized and trusted platform to connect with peers for academic assistance, skill-sharing, and collaborative learning. Current platforms focus on social interaction rather than structured student growth, creating a need for a dedicated peer-learning ecosystem
# PeerLearn

PeerLearn is a collaborative learning platform where students can:

- connect with peers
- join study groups
- take AI-powered quizzes
- customize profiles
- chat and video call

## Features

- Responsive UI
- Dark Mode
- AI Quiz Engine
- Peer Chat
- Video Calling
- Gamified XP System

## Tech Stack

- HTML
- CSS
- JavaScript
